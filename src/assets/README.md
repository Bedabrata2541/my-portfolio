## Download Instructions

1. **Right-click each image URL** and select "Open in new tab"
2. **Right-click on the opened image** and select "Save image as..."
3. **Save the image** in this `/assets` folder with the corresponding filename
4. **Update the import paths** in your local project files to reference these assets

## Usage in Local Development

Once downloaded, update the import statements in `/src/components/Projects.tsx`:

```typescript
// Replace the figma:asset imports with local asset imports
import pragatiImage from '../assets/pragati-image.png';
import sanjeevaniImage from '../assets/sanjeevani-image.png';
import beyondPoweredImage from '../assets/beyondpowered-image.png';
import trinetraImage from '../assets/trinetra-image.png';
```

## Image Specifications

- **Format**: PNG or JPG (both work)
- **Recommended Size**: 1080px width (already optimized in URLs)
- **Aspect Ratio**: 16:9 recommended for consistent card display
- **Quality**: High quality from Unsplash (professional stock photography)
